# Проект
Этот учебный репозиторий содержит два связанных микросервиса - Payment Service и Async Payment Processor (Kafka, Docker, CI/CD, Spring Boot, REST, PostgreSQL)

## Цель и задачи
Шаг за шагом я пришел к пониманию принципов работы микросервисов, основы  интеграционного и Unit-тестирования,  организации безопасности,  асинхронной обработки платежей и сообщений.

### Структура проекта
- Основа - _Spring Boot, Hibernate._
- База данных - _PostgreSQL, Spring Data, Liquibase,Postman_
- Тестирование - _JUnit, Mockito_
- Организация безопасности - _Spring Boot Security, KeyCloak_
- Асинхронная обработка - _Kafka, RabbitMQ_
